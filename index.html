📖 나의 서재 — 도움말

✨
현재 버전 v1.6
HL 전용 독서 기록 앱
모든 건의 및 문의 사항은 트위터 계정 @HLismy1ife의 바이오 링크 익명 질문함으로 주세요!
⚠️ HL 외 BL&GL 장르로 사용하지 마세요. 발견 즉시 폐쇄합니다.
🗂 주요 기능
📚
책 추가 / 수정
표지 이미지, 장르, 책 종류, 이용가 등급, 구매 플랫폼·금액, 소재/관계 태그, 남주·여주 정보, 줄거리, 총평, 별점, 발췌 무한 추가
🏷
소재 / 관계 태그
소재(#계약결혼 #환생)와 관계(#소꿉친구 #혐관)를 분리해서 입력. 상세보기에서 구분되어 표시
💙🩷
남주 / 여주 정보
이름·나이·유형·소개 각각 입력. 상세보기에서 카드 형태로 표시
📊
통계
완독·읽는중·읽고싶어요 현황, 총 소비 금액, 평균 독서 시간, 월별 구매 현황 그래프, 장르·별점·플랫폼 분포
📅
완독 캘린더
완독일 기준으로 달력에 표시. ◀ ▶로 월 이동, 완독한 날에 파란 점. 옆에 이달의 완독 목록과 독서 목표 게이지
💌
추천 카드
상세보기에서 "추천 카드" 클릭 → 4가지 테마, 한 줄 멘트 입력 → 고화질 PNG로 저장
⇅
JSON 백업 / 복원
헤더의 ⇅ 버튼으로 전체 데이터를 JSON으로 저장하거나 불러올 수 있음. GitHub Pages에 올릴 때 데이터 이전에 활용
✏️
서재 이름 변경
헤더 로고 클릭 → 이모지 40개 중 선택 + 이름 직접 입력. 브라우저에 저장돼서 새로고침해도 유지
💡 이용 방법
1헤더 + 추가 버튼으로 책 등록
2카드 클릭하면 상세보기, 상세보기에서 ✏️ 수정으로 편집
3장르 필터·상태 필터·정렬·검색으로 원하는 책 바로 찾기
4통계 탭에서 내 독서 패턴 한눈에 확인
5완독 캘린더에서 독서 목표 설정 후 진행률 체크
6데이터는 브라우저 IndexedDB에 자동 저장 — 용량 제한 없음, 정기적으로 JSON 백업 권장
